## 📚 仓库简介

这是一个专注于 **小升初自然科学英文阅读** 的笔记仓库，精选适合 11-13 岁学生的英文科普文章，搭配：
- 📝 Markdown 详细笔记（中英双语术语对照）
- 🧠 Anki 记忆卡（即学即背）
- 🔗 Obsidian 双链笔记（知识网络）
- 📊 Mermaid 流程图（可视化理解）

## 🚀 快速开始

### 1. 本地阅读
```bash
git clone https://github.com/你的用户名/英语学习兔.git
cd 英语学习兔/小升初英语
```

### 2. 导入 Anki
```
Anki → 文件 → 导入 → 选择 99-Anki卡包/*.csv → 确认
```

### 3. Obsidian 双链
```
Obsidian → 打开仓库 → 启用图形视图
```
