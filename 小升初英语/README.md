## 🧠 学习方法

### 📖 三色标注阅读法

| 颜色 | 标记内容 | 示例 |
|------|---------|------|
| 🔵 蓝色 | 科学概念 | pollination, gravity |
| 🟢 绿色 | 过程描述 | How bees carry pollen... |
| 🟡 黄色 | 专业术语 | ecosystem service, centrifugal force |

### 🔄 艾宾浩斯复习节奏

| 复习节点 | 时间点 | 动作 |
|---------|-------|------|
| 第1次 | 当天学习后 | 通读笔记，尝试背诵 |
| 第2次 | 第2天 | Anki复习所有卡片 |
| 第3次 | 第4天 | 默写关键词英文 |
| 第4次 | 第7天 | 口头复述全文（中英双语） |
| 第5次 | 第14天 | 写一段自己的英文总结 |
